# Event Scheduler

A simple web-based Event Scheduler application that allows users to create and manage events efficiently.

## Features

- Add new events
- Edit existing events
- Delete events
- Search events
- Calendar view
- Animated background UI
- Responsive design

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Calendar

## Installation

Clone the repository

git clone https://github.com/GugulothNandi/-event-scheduling-system.git

Navigate to project folder

cd -event-scheduling-system

Install dependencies

npm install

Run the project

npm run dev
